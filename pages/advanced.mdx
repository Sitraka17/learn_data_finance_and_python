Python
import yfinance as yf
import pandas as pd
import matplotlib.pyplot as plt

# 1. Define the asset and grab the data
ticker = "AAPL" # Apple's stock ticker
print(f"Fetching real-time historical data for {ticker}...")

# Download the last 2 years of daily stock prices
stock_data = yf.download(ticker, period="2y")

# 2. Core Financial Math: Returns and Moving Averages
# Calculate daily percentage returns (How much it went up/down each day)
stock_data['Daily Return'] = stock_data['Close'].pct_change()

# Calculate the 50-day and 200-day Simple Moving Averages (SMA)
# This is a classic indicator used in algorithmic trading
stock_data['SMA_50'] = stock_data['Close'].rolling(window=50).mean()
stock_data['SMA_200'] = stock_data['Close'].rolling(window=200).mean()

# Clean up empty data created by the rolling averages
stock_data.dropna(inplace=True)

# 3. Output some basic quantitative stats
print("\n--- Quick Quantitative Stats ---")
print(f"Average Daily Return: {stock_data['Daily Return'].mean():.4f}")
print(f"Daily Volatility (Risk): {stock_data['Daily Return'].std():.4f}")

# 4. Visualize the data like a pro
plt.figure(figsize=(12, 6))

# Plot the actual closing price
plt.plot(stock_data.index, stock_data['Close'], label=f'{ticker} Close Price', color='blue', alpha=0.5)

# Plot the moving averages
plt.plot(stock_data.index, stock_data['SMA_50'], label='50-Day SMA', color='orange', linewidth=2)
plt.plot(stock_data.index, stock_data['SMA_200'], label='200-Day SMA', color='red', linewidth=2)

# Formatting the chart
plt.title(f'{ticker} - Price & Moving Averages')
plt.xlabel('Date')
plt.ylabel('Price (USD)')
plt.legend()
plt.grid(True)
plt.show()
Why This Code Matters (The Breakdown)
yfinance: This is your free data pipeline. Instead of paying for Bloomberg terminals, this pulls data straight from Yahoo Finance.

.pct_change(): This is Pandas doing the heavy lifting. In one line of code, it calculates the daily returns for the entire two-year dataset.

.rolling(window=50).mean(): This calculates the Moving Average. Quants use this to smooth out daily price noise and spot actual trends. When a 50-day SMA crosses above a 200-day SMA, it's called a "Golden Cross" (a bullish trading signal).
